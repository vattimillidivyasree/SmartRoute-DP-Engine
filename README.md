# 🧠 SmartRoute – DP Engine
<br/>
SmartRoute is a small project where I explored how to solve pathfinding and grid-based problems using Dynamic Programming. Instead of relying on brute-force approaches, I focused on writing optimized solutions using memoization and tabulation.
<br/>

## 🚀 Features
<br/>
• Solves common pathfinding problems using DP <br/>
• Shows both recursive and optimized approaches <br/>
• Focuses on writing efficient and clean logic <br/>
• Easy to understand and extend <br/>

## ⚙️ Tech Stack
<br/>
JavaScript (Node.js) <br/>
